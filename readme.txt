== Description ==

r47Twitter allows you to show your last Tweets from Twitter

== Installation ==

Zip:
Upload the r47Twitter plugin to your blog

Activate it
Add the Widget where you want

Git:
Clone the r47Twitter-Plugin from github into the /wp-content/plugin/r47Twitter directory

git clone git://github.com/robo47/r47Twitter.git wp-content/plugin/r47Twitter

Activate it
Add the Widget where you want
